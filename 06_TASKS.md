# TASKS (AI 개발 파트너용 프롬프트 설계서)
## K모델크루 아카데미 홈페이지

**문서 버전:** v1.0  
**작성일:** 2026년 1월 7일  
**프로젝트 식별자:** KMODEL-2026  
**관련 문서:** PRD, TRD, User Flow, Database Design, Design System

---

## 문서 개요

이 문서는 AI 코딩 파트너(Claude, Cursor 등)가 즉시 협업을 시작할 수 있도록 구체화된 단계별 개발 태스크를 제공합니다. 각 태스크는 자연어로 작성되어 있으며, 관련 문서를 참조하여 작업합니다.

**마일스톤 구조:**
- M0: 프로젝트 초기화
- M1: 핵심 UI 및 디자인 시스템
- M2: 데이터베이스 및 백엔드
- M3: 핵심 기능 개발 (MVP)
- M4: 외부 서비스 연동
- M5: 관리자 페이지
- M6: 테스트 및 배포
