# PRD (제품 요구사항 정의서)
## K모델크루 아카데미 홈페이지

**문서 버전:** v1.0  
**작성일:** 2026년 1월 7일  
**프로젝트 식별자:** KMODEL-2026

---

## 1. 개요

### 1.1 프로젝트명
**K모델크루 아카데미** 공식 홈페이지

### 1.2 한 줄 정의
> "시니어 세대가 모델의 꿈을 실현할 수 있도록 전문 교육과 활동 기회를 제공하는 아카데미"의 온라인 얼굴

### 1.3 궁극적 목표
**제2의 인생에서 자신감과 활력을 되찾게 해주는 것** — 단순히 정보를 전달하는 홈페이지가 아니라, 방문자가 "나도 할 수 있겠다"는 용기를 얻고 첫 발을 내딛게 만드는 디지털 공간

---

## 2. 문제 정의

### 2.1 현재 상황
K모델크루 아카데미는 인스타그램(@kmodel_crew)을 통해 학원을 홍보하고 있으나, 다음과 같은 한계에 직면해 있습니다:

| 문제 | 영향 |
|------|------|
| 커리큘럼, 수강료, 강사진 정보를 체계적으로 보여줄 공간 부재 | 잠재 수강생이 필요한 정보를 얻기 어려움 |
| 경쟁 학원들은 홈페이지가 있음 | 상대적으로 신뢰도가 낮아 보임 |
| 인스타그램만으로는 전문성 전달에 한계 | "제대로 된 학원인가?" 의문 발생 |

### 2.2 해결 가설
> "전문적이고 신뢰감 있는 홈페이지가 있으면, 인스타그램에서 유입된 잠재 수강생의 상담 전환율이 높아질 것이다."

## 3. 사용자 페르소나

### 3.1 주요 페르소나: "새로운 도전을 찾는 김영숙 님"

| 항목 | 내용 |
|------|------|
| **나이/성별** | 55세 여성 |
| **상황** | 자녀가 독립하고, 빈 둥지 증후군을 겪으며 "나를 위한 시간"을 찾고 있음 |
| **관심사** | 패션, 자기계발, TV에서 본 시니어 모델들의 활약 |
| **고민** | "이 나이에 시작해도 될까?", "어디서 배우지?", "돈만 버리는 건 아닐까?" |
| **기대** | 체계적인 교육, 실제 활동 기회, 같은 꿈을 꾸는 동료들과의 만남 |
| **기기 사용** | 스마트폰 중심 (70%), 가끔 집에서 PC (30%) |

### 3.2 사용자 여정
```
인스타그램에서 K모델크루 게시물 발견
    ↓
"여기 괜찮아 보이는데?" → 프로필의 홈페이지 링크 클릭
    ↓
홈페이지에서 소속 모델 포트폴리오 구경
    ↓
"와, 저분들도 여기서 배웠구나" → 아카데미 프로그램 확인
    ↓
커리큘럼, 수강료, 강사진 정보 확인
    ↓
"한번 상담받아볼까?" → 상담 신청
```

## 4. 사용자 스토리

### 4.1 핵심 기능별 사용자 스토리

#### FEAT-1: 회사 소개
> "나는 **잠재 수강생**으로서, **학원의 철학과 대표 소개, 연혁**을 보고 싶다. **이 학원이 믿을 만한 곳인지 판단**하기 위해서이다."

**인수 조건:**
- 학원 설립 배경과 철학이 명확히 전달됨
- 대표(원장) 소개 및 프로필이 노출됨
- 학원의 차별점이 한눈에 파악됨
- 시니어 친화적 폰트 크기와 간결한 문장

#### FEAT-2: 아카데미 프로그램 안내
> "나는 **잠재 수강생**으로서, **어떤 과정이 있고, 무엇을 배우며, 비용이 얼마인지** 알고 싶다. **등록 여부를 결정**하기 위해서이다."

**인수 조건:**
- 커리큘럼(과정명, 기간, 내용)이 체계적으로 정리됨
- 강사진 소개 및 프로필이 노출됨
- 수강료 정보가 명시됨 (또는 상담 안내)
- 수업 일정/시간표 확인 가능

#### FEAT-3: 소속 모델 포트폴리오 (MVP 핵심)
> "나는 **잠재 수강생**으로서, **이 학원에서 배운 분들의 실제 활동 모습**을 보고 싶다. **나도 저렇게 될 수 있다는 확신**을 얻기 위해서이다."

**인수 조건:**
- 소속 모델들의 프로필 사진과 기본 정보 노출
- 활동 이력(패션쇼, 화보, 방송 등) 확인 가능
- 고화질 이미지로 전문적인 인상 전달
- 모델별 상세 페이지 또는 갤러리 형태

#### FEAT-4: 상담 신청
> "나는 **잠재 수강생**으로서, **궁금한 점을 문의하거나 상담을 신청**하고 싶다. **다음 단계로 나아가기** 위해서이다."

**인수 조건:**
- 간단한 상담 신청 폼 (이름, 연락처, 문의 내용)
- 신청 완료 시 알림톡/문자 자동 발송 (학원 측에)
- 전화번호, 이메일, 위치 정보 명시
- 카카오맵/네이버맵 연동으로 찾아오는 길 안내

#### FEAT-5: 인스타그램 피드 연동
> "나는 **방문자**로서, **학원의 최근 활동과 소식**을 홈페이지에서도 보고 싶다. **활발하게 운영되는 학원인지 확인**하기 위해서이다."

**인수 조건:**
- 인스타그램 최신 게시물이 자동으로 표시됨
- 인스타그램 계정으로 연결되는 링크 제공
- 실시간 또는 주기적 업데이트

## 6. 범위 정의

### 6.1 MVP 범위 (첫 버전에 포함)
| 우선순위 | 기능 | 식별자 |
|----------|------|--------|
| 1 | 소속 모델 포트폴리오 | FEAT-3 |
| 2 | 회사 소개 | FEAT-1 |
| 3 | 아카데미 프로그램 안내 | FEAT-2 |
| 4 | 상담 신청 (폼 + 알림) | FEAT-4 |
| 5 | 인스타그램 피드 연동 | FEAT-5 |
| 6 | 찾아오는 길 (지도) | FEAT-4 내 포함 |

### 6.2 범위 외 (Non-goals) — 지금은 하지 않음
| 기능 | 지연 사유 | 향후 계획 |
|------|-----------|-----------|
| 회원가입/로그인 | 현재 필요 없음 | 커뮤니티 도입 시 검토 |
| 온라인 스토어 | MVP 범위 초과 | 성장 후 확장 (Phase 2) |
| 온라인 강의 판매 | 현재 오프라인 중심 | 장기 검토 |
| 게시판/커뮤니티 | 운영 리소스 부족 | 수강생 증가 시 검토 |
| 다국어 지원 | 국내 타겟 우선 | 해외 진출 시 검토 |

## 7. 가정과 리스크

### 7.1 핵심 가정
| 가정 | 검증 방법 |
|------|-----------|
| 인스타그램이 주요 유입 경로일 것이다 | GA에서 유입 경로 분석 |
| 50~60대도 모바일 웹을 불편 없이 사용할 것이다 | 사용성 테스트, 이탈률 모니터링 |
| 소속 모델 포트폴리오가 가장 설득력 있을 것이다 | 페이지별 체류 시간 비교 |
| 홈페이지가 있으면 신뢰도가 높아질 것이다 | 상담 신청 시 "어떻게 알게 되셨나요" 문항 |

### 7.2 리스크 보드

| 리스크 | 영향 | 가능성 | 완화 전략 | 트리거 |
|--------|------|--------|-----------|--------|
| 시니어가 홈페이지를 어려워함 | 높음 | 중간 | 시니어 친화적 UI, 큰 글씨, 단순한 메뉴 | 이탈률 70% 초과 시 |
| 인스타 연동 API 제한 | 중간 | 중간 | 수동 갤러리 업로드 대안 준비 | API 차단 시 |
| 콘텐츠 업데이트 부담 | 중간 | 높음 | 쉬운 관리자 페이지, 최소 업데이트 구조 | 3개월 이상 미업데이트 시 |
| 경쟁사 대비 차별화 부족 | 중간 | 낮음 | 실제 수강생 후기, 생생한 활동 사진 강조 | 전환율 0.5% 미만 시 |

## 8. 디자인 원칙

### 8.1 비주얼 방향
- **고급스럽고 세련된** 블랙 & 화이트 톤
- **패션 매거진** 같은 분위기
- **시니어 친화적** — 화려하지만 복잡하지 않게

### 8.2 톤 앤 보이스
- **격식체 + 고급스러운 톤**
- 예시: "새로운 도약을 함께합니다", "당신의 가능성을 발견하세요"
- 피해야 할 것: 과도한 감탄사, 반말, 유행어

### 8.3 접근성 원칙
- 최소 폰트 크기 16px 이상
- 버튼/터치 영역 최소 44px
- 고대비 색상 조합
- 단순하고 예측 가능한 네비게이션

## 9. 실험 → 학습 → 다음 가설

### 9.1 MVP 출시 후 실험 계획

| 실험 | 관측 지표 | 학습 목표 |
|------|-----------|-----------|
| 포트폴리오 페이지를 랜딩으로 설정 | 체류 시간, 다음 페이지 이동률 | 첫인상으로 포트폴리오가 효과적인가? |
| 상담 신청 버튼 위치 A/B 테스트 | 클릭률 | 어느 위치가 전환에 효과적인가? |
| 수강료 공개 vs 상담 유도 | 상담 신청 수 | 가격 공개가 전환에 도움이 되는가? |

2. 수강생 후기 섹션 강화
3. 뉴스/미디어 섹션 추가 (활동 소식)
